TradingView List

This website lists symbol, exchange, and screener supported by the python library tradingview-ta. The database isn't complete, so please open an issue on GitHub if you can't find your country. Please note that indices are not supported by tradingview-ta, this is a limitation from TradingView's API, not the library.
